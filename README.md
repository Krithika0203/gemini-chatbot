**Gemini Pro Nexus** is a premium, high-performance neural interface built with Streamlit and Google's latest Generative AI models. It features a sleek "Glassmorphism" UI, real-time creative controls, and advanced chat memory.

---

## ✨ Key Features

- **2026 Engine Support**: Fully compatible with the Gemini 3.1 series (Pro, Flash, and Flash-Lite).
- **Glassmorphism UI**: A custom-styled, professional dark-mode interface with blur effects and radial gradients.
- **Dynamic Configuration**: Adjust creativity (Temperature) and switch intelligence engines on the fly via the sidebar.
- **Neural Link Memory**: Persistent session-state chat history that "remembers" the conversation flow.
- **Secure Architecture**: Built-in support for environment variables to keep your API keys private.

## 🚀 Quick Start

### 1. Prerequisites
- Python 3.9 or higher
- A Google Gemini API Key (Get one at [Google AI Studio](https://aistudio.google.com/))

### 2. Installation
Clone the repository and install the dependencies:

```bash
git clone [https://github.com/YOUR_USERNAME/gemini-pro-nexus.git](https://github.com/YOUR_USERNAME/gemini-pro-nexus.git)
cd gemini-pro-nexus
pip install -r requirements.txt
